# Deploy to Vercel — Skotjänst

## Option A — CLI

```powershell
cd C:\Jarvis\projects\business-2-skotjanst
npx vercel --yes
npx vercel --prod --yes
```

## Option B — GitHub → Vercel

1. Push repo per [GITHUB_SETUP.md](./GITHUB_SETUP.md)
2. [vercel.com/new](https://vercel.com/new) → **Import** `business-2-skotjanst`
3. Framework: **Next.js** (auto)
4. Click **Deploy**

## Vercel Buttons

1. **Add New…** → **Project**
2. **Import** repo
3. **Deploy**
4. **Visit**
